# Clip Extraction & Mathematical Attribution Model

**Date:** 2026-04-26

## 1. Exact Algorithm for Clip Extraction & Scoring
The goal is to automatically find the most engaging 15-60 second clips from long-form video/audio based on the Natural Center.

**Algorithm:**
1. **Transcription & Diarization:** Run Whisper over the media. Get timestamps for every word.
2. **Pathos Scoring:** Calculate the "Energy Delta" (ΔE) per 10-second window. `ΔE = (Volume_Variance + Pitch_Frequency_Shift) * Speaking_Rate`.
3. **Logos/Lexical Scoring:** For every sentence, calculate a vector embedding. Compute cosine similarity between the sentence vector and the `Natural_Center_Core_Vector`.
4. **Composite Score:** `Clip_Score = (w1 * Pathos_Score) + (w2 * Lexical_Score)`.
5. **Boundary Identification:** When a rolling 30-second window exceeds `Threshold_Alpha`, mark start. Expand boundaries until `Clip_Score` drops below `Threshold_Beta` or a natural pause occurs.
6. **Output:** A JSON array of `{start_time, end_time, score, primary_keyword}`.

## 2. Mathematical Attribution Model
How do we prove a specific clip caused a conversion?

**The Multi-Touch Markov Chain Attribution:**
We do not use last-click. We model the customer journey as a Markov chain where each state is a piece of content viewed.

- Let $S$ be the set of all clips.
- Let $P(i, j)$ be the transition probability from viewing clip $i$ to viewing clip $j$.
- Let $C$ be the conversion state.
- **Removal Effect:** To find the true value of clip $k$, calculate the probability of conversion with all nodes intact ($P_C$). Then remove node $k$ from the graph and recalculate the probability of conversion ($P_{C \setminus k}$).
- **Attribution Value:** $V(k) = 1 - \frac{P_{C \setminus k}}{P_C}$

This mathematically proves the *incremental* value of every single atom generated by the system. It replaces guessing with absolute, dissertation-grade certainty.
