# XVI. ATTRIBUTION MODEL

The mathematically correct attribution model is hierarchical, weighted, time-decayed, and platform-normalized. fileciteturn0file14

Lineage:

```text
Asset → Fragment → GeneratedUnit → PublishEvent → PerformanceObservation → AudienceStateTransition / EconomicOutcome
```

Rollup rule:

```text
Attr(asset) = Σ publish_events [ lineage_weight * platform_normalized_value * time_decay ]
```

Economic layer:

```text
NetAssetReturn = attributed_value - allocated_cost
ROI = (attributed_value - allocated_cost) / allocated_cost
AssetYieldMultiplier = attributed_value / allocated_cost
```

This is the correct production accounting doctrine.
