## 1. $EXP_NC_ALIGNMENT_LIFT

### Purpose
Test whether **Natural Center alignment causally improves performance**

### Hypothesis

```text
H1: Higher NC_score → higher audience retention + follower conversion
```

### Design

```yaml
treatment:
  units with NC_score ≥ 0.75

control:
  units with 0.55 ≤ NC_score < 0.65

randomization_unit: publish_event
stratification:
  platform
  content_type
```

### Metrics

```yaml
primary:
  retention_rate
  follower_conversion_rate

secondary:
  watch_time
  engagement_rate
```

### Estimator

\[
Lift_{NC} = E[Y | NC_{high}] - E[Y | NC_{mid}]
\]

### Implementation

```python
def assign_nc_experiment(unit):
    score = nc_score(unit, NC)
    if score >= 0.75:
        return "treatment"
    elif 0.55 <= score < 0.65:
        return "control"
    else:
        return "exclude"
```

### Success Criteria

```text
Lift_NC ≥ +8% retention OR +5% conversion
p_value < 0.05
```
