### Formal

\[
\forall u \in U,\quad NC\_match(u, NC) \ge \tau_{NC}
\]

### Contract

```yaml
id: AXIOM_01_IDENTITY
type: hard_invariant
enforcement_stage: pre_publish
blocking: true
```

### API


### Enforcement

```python
if NC_score(u, NC) < NC_THRESHOLD:

```

### Schema Binding

