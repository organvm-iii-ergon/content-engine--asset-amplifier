## Exact extraction algorithm

I would formalize the Natural Center subsystem as `ALG_NC_01`.

### Stage 0: corpus assembly

Input corpus \(C\) should be tenant- and brand-scoped:

\[
C = C_v \cup C_i \cup C_a \cup C_t \cup C_h
\]

where:

\[
C_v = \text{video assets}
\]
\[
C_i = \text{image / render assets}
\]
\[
C_a = \text{audio assets}
\]
\[
C_t = \text{text / voice corpus}
\]
\[
C_h = \text{historical high-performing outputs, if available}
\]

If only one premium asset exists, the sources still allow `NATURAL_CENTER_MIN_ASSETS=1`, so bootstrap from a single asset is valid, but confidence must be lower and human review should remain enabled. ŅłĆfileciteŅłéturn1file2Ņłü

### Stage 1: multimodal feature extraction

For each asset \(x \in C\), compute feature families.

For visual inputs:

\[
\phi_v(x)=\langle color,\ composition,\ framing,\ motion,\ texture,\ saliency,\ object\_presence\rangle
\]

For audio inputs:

\[
\phi_a(x)=\langle rms,\ spectral\_flux,\ tempo,\ silence,\ emphasis,\ prosody\rangle
\]

For text and speech:

\[
\phi_t(x)=\langle lexical\_density,\ sentence\_length,\ rhetoric,\ polarity,\ modality,\ authority,\ abstraction\rangle
\]

This matches the source extraction logic: color histograms, motion stats, composition maps, lexical density, sentence-length distribution, rhetoric markers, semantic embeddings, and symbolic recurrence. ŅłĆfileciteŅłéturn2file0Ņłü

### Stage 2: semantic embedding and clustering

Embed all semantic-bearing units:

\[
e_i = \text{Embed}(u_i)
\]

where \(u_i\) may be frame captions, transcript spans, textual paragraphs, OCR snippets, or historical post bodies.

Cluster embeddings:

\[
\mathcal{K} = \text{Cluster}(\{e_i\})
\]

For each cluster \(k\), compute:

\[
mass(k)=|k|
\]
\[
cohesion(k)=\frac{1}{|k|}\sum_{e_j\in k}\cos(e_j,\mu_k)
\]
\[
stability(k)=presence(k)\cdot cohesion(k)
\]

Select top thematic clusters:

\[
T = TopK(\mathcal{K},\ score = stability,\ k=3..5)
\]

This is directly consistent with the source rule that candidate themes are clustered first and then only stable top-\(k\) themes are retained. ŅłĆfileciteŅłéturn2file0Ņłü

### Stage 3: aesthetic centroid

From visual features across corpus, compute a weighted centroid:

\[
A = \frac{\sum_i w_i \phi_v(x_i)}{\sum_i w_i}
\]

Weights should privilege canonical assets over derivatives:

\[
w_i =
\begin{cases}
1.0 & \text{hero asset} \\
0.6 & \text{supporting asset} \\
0.4 & \text{historical output}
\end{cases}
\]

Decompose \(A\) into subcomponents:

\[
A=\langle palette,\ framing,\ motion,\ texture,\ contrast,\ density\rangle
\]

This stabilizes the visual attractor rather than overfitting to any single frame.

### Stage 4: tonal centroid

From linguistic and speech features, compute tonal axes:

\[
\Tau=\langle formal,\ expressive,\ emotional,\ authoritative,\ playful\rangle
\]

Operationally:

\[
\Tau = \frac{\sum_i w_i \phi_t(x_i)}{\sum_i w_i}
\]

Then normalize each axis to \([0,1]\).

This matches the source notion of a tonal vector and the later use of tonal alignment as a constraint variable in fragment and output scoring. ŅłĆfileciteŅłéturn2file0Ņłéturn1file13Ņłü

### Stage 5: narrative bias extraction

Define a narrative pattern function over semantic units:

\[
n(u)=\langle transformation,\ tension,\ reveal,\ aspiration,\ proof,\ authority\rangle
\]

Aggregate across corpus:

\[
N = \frac{\sum_i w_i n(u_i)}{\sum_i w_i}
\]

This gives you a persistent bias profile. For example, some brands resolve toward proof and authority; others toward transformation and aspiration.

### Stage 6: symbolic marker discovery

Let \(m_j\) be recurring symbolic candidates, visual or textual. For each candidate compute:

\[
symbolic\_score(m_j)=frequency(m_j)\cdot distinctiveness(m_j)\cdot consistency(m_j)
\]

Accept into symbolic marker set \(S\) if:

\[
symbolic\_score(m_j)\ge \tau_{symbol}
\]

The source env contract already suggests a production threshold, `SYMBOLIC_MARKER_MIN_SCORE=0.70`. ŅłĆfileciteŅłéturn1file2Ņłü

### Stage 7: negative-space derivation

This is critical. The source set correctly frames negative space as ŌĆ£what the system must never produce.ŌĆØ ŅłĆfileciteŅłéturn0file32Ņłü

Formally, define the universal feature pool \(U\) for your generation domain. Then:

\[
\Omega = U \setminus H
\]

where \(H\) is the set of consistently present, accepted, or high-confidence brand-aligned features.

In practice, compute a prohibited-feature score:

\[
forbid(z)=absence(z)\cdot incompatibility(z)\cdot drift\_risk(z)
\]

and include \(z\) in negative space if:

\[
forbid(z)\ge \tau_{\Omega}
\]

Examples include meme tonality, bright-saturated palette, slang, jump-cut overload, stock-motivation rhetoric, or comedic framing.

### Stage 8: invariant induction

Convert stable feature patterns into enforceable rules.

For each candidate feature \(q\), compute:

\[
stability(q)=presence\_across\_assets(q)\cdot consistency\_across\_assets(q)
\]

This stability rule is already specified in the source material. ŅłĆfileciteŅłéturn2file0Ņłü

Accept as invariant if:

\[
stability(q)\ge \tau_{inv}
\]

Then emit rule forms such as:

```text
all outputs must express at least one thematic_core element
all captions must remain within tonal distance threshold
all visual outputs must remain inside palette bounds
no output may violate negative_space
```

### Stage 9: mutation-bound estimation

Mutation bounds determine how far outputs may vary while remaining ŌĆ£of the same identity.ŌĆØ

For each feature family \(d\), estimate historical variance:

\[
\sigma_d^2 = Var_d(C)
\]

Set allowed mutation bound:

\[
\mathcal{M}_d = [\mu_d - \lambda\sigma_d,\ \mu_d + \lambda\sigma_d]
\]

with lower \(\lambda\) for identity-critical dimensions and higher \(\lambda\) for expressive dimensions.

A reasonable first partition is:

```text
palette / tone / symbolic recurrence: low mutation
hook structure / pacing / CTA framing: medium mutation
platform packaging details: high mutation
```

### Stage 10: embedding synthesis

Finally, build a center embedding:

\[
E = \text{Fuse}(Embed(T), Embed(A), Embed(\Tau), Embed(N), Embed(S))
\]

where `Fuse` may be concatenation followed by learned projection, or weighted averaging into a shared latent space.

This produces a single vector against which downstream items can be scored for Natural Center similarity.

## Confidence model

The missing precision in the earlier drafts is confidence accounting. You need it.

Define:

\[
conf(NC)=g(n_{assets}, modality\_coverage, feature\_stability, review\_status)
\]

A simple V1 form:

\[
conf(NC)=0.35\cdot assets\_score + 0.25\cdot modality\_score + 0.25\cdot stability\_score + 0.15\cdot review\_score
\]

This matters because the sources allow single-asset bootstrap, but single-asset NC should not claim the same certainty as a mature multi-asset corpus. ŅłĆfileciteŅłéturn1file2Ņłü

## Downstream enforcement

Once extracted, the Natural Center must act as a gate.

For any fragment or generated unit \(y\), compute:

\[
NC\_match(y)=\alpha_1\cdot thematic\_alignment
+\alpha_2\cdot tonal\_alignment
+\alpha_3\cdot aesthetic\_match
+\alpha_4\cdot symbolic\_marker\_density
\]

Those component scores are already defined in the source scoring layer. ŅłĆfileciteŅłéturn1file13Ņłéturn2file0Ņłü

Then apply hard gating:

```text
if violates_negative_space(y, NC): reject
if tonal_alignment(y) < Žä_tone: reject
if NC_match(y) < Žä_nc: reject
```

This is exactly how the source system already treats negative space and tonal alignment in fragment gating. ŅłĆfileciteŅłéturn1file13Ņłü

## Final compact specification

The formal extraction contract is:

\[
NC = ALG\_NC\_01(C)
\]

with

\[
ALG\_NC\_01 =
derive\_themes
\circ derive\_aesthetics
\circ derive\_tone
\circ derive\_narrative
\circ derive\_symbols
\circ derive\_negative\_space
\circ induce\_invariants
\circ estimate\_mutation\_bounds
\circ encode
\]

and acceptance criterion:

\[
NC\ \text{is valid iff}\ conf(NC)\ge \tau_{conf}
\]

while any downstream output \(y\) is valid iff:

\[
NC\_match(y)\ge \tau_{nc}
\quad\land\quad
drift(y)\le \tau_{drift}
\quad\land\quad
\neg violates\_negative\_space(y)
\]

That is the correct formalization.
