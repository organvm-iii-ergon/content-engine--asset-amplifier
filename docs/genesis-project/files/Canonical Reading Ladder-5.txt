## I.6 Reference implementation

```python
from typing import TypedDict, List, Dict


class NCScoreResult(TypedDict):
    total_score: float
    thematic_alignment: float
    aesthetic_alignment: float
    tonal_alignment: float
    narrative_alignment: float
    symbolic_alignment: float
    negative_space_violation: float
    invariant_pass: bool
    hard_fail_reasons: List[str]
    score_version: str
    nc_id: str
    nc_version: int


def _clamp(value: float, lo: float = 0.0, hi: float = 1.0) -> float:
    return max(lo, min(hi, value))


def nc_score(candidate: dict, nc: dict, mode: str = "unit") -> NCScoreResult:
    thematic_alignment = _clamp(candidate["semantic_features"]["thematic_alignment"])
    aesthetic_alignment = _clamp(candidate["visual_features"]["aesthetic_alignment"])
    tonal_alignment = _clamp(candidate["tonal_features"]["tonal_alignment"])
    narrative_alignment = _clamp(candidate["narrative_features"]["narrative_alignment"])
    symbolic_alignment = _clamp(candidate["symbolic_features"]["symbolic_alignment"])
    negative_space_violation = _clamp(
        candidate["negative_space_features"]["negative_space_violation"]
    )

    hard_fail_reasons: List[str] = []

    if tonal_alignment < nc["mutation_bounds"].get("min_tonal_alignment", 0.60):
        hard_fail_reasons.append("tonal_alignment_below_floor")

    if negative_space_violation > nc["mutation_bounds"].get("max_negative_violation", 0.20):
        hard_fail_reasons.append("negative_space_violation")

    invariant_pass = len(hard_fail_reasons) == 0

    total_score = _clamp(
        0.22 * thematic_alignment
        + 0.18 * aesthetic_alignment
        + 0.18 * tonal_alignment
        + 0.14 * narrative_alignment
        + 0.18 * symbolic_alignment
        - 0.20 * negative_space_violation
    )

    return NCScoreResult(
        total_score=total_score,
        thematic_alignment=thematic_alignment,
        aesthetic_alignment=aesthetic_alignment,
        tonal_alignment=tonal_alignment,
        narrative_alignment=narrative_alignment,
        symbolic_alignment=symbolic_alignment,
        negative_space_violation=negative_space_violation,
        invariant_pass=invariant_pass,
        hard_fail_reasons=hard_fail_reasons,
        score_version="nc_score_v1",
        nc_id=nc["nc_id"],
        nc_version=nc["version"],
    )
```

## II.4 Master equation

\[
Score(x,p,a,t)=
w_I I(x)
+ w_N N(x,NC)
+ w_P P(x,p,t)
+ w_A A(x,a)
- w_R R(x)
\]

where:

```text
I = intrinsic quality score
N = NC alignment score
P = platform-fit score
A = audience-fit score
R = risk / penalty score
```

This is the exact scoring unification the previous artifact called for. fileciteturn5file26

## II.5 Component definitions

### Intrinsic score

Used mainly for fragment and unit quality.

\[
I(x)=
\sum_i \omega_i x_i
\]

Typical inputs come from the extraction stack:

```text
visual_saliency
motion_energy
hook_density
speech_clarity
semantic_novelty

```

These are already specified in the algorithmic frame. fileciteturn5file13

### NC alignment score

\[
N(x,NC)=NCScore(x,NC)
\]

### Platform fit score

\[
P(x,p,t)=
a_1 duration\_fit
+a_2 opening\_pattern\_fit
+a_3 caption\_density\_fit
+a_4 format\_weight\_match
+a_5 current\_platform\_preference\_match
\]

This follows the platform profile doctrine and platform-state model. fileciteturn5file1 fileciteturn5file13

### Audience fit score

\[
A(x,a)=
b_1 retention\_potential
+b_2 identity\_binding
+b_3 CTA\_fitness
+b_4 seriality\_support
\]

This is justified by the audience-state machine and audience engine layer. fileciteturn5file9

### Risk penalty

\[
R(x)=
r_1 negative\_space\_violation
+r_2 redundancy
+r_3 crop\_risk
+r_4 legibility\_risk
+r_5 drift\_risk
\]

## II.7 Reference implementation

```python
def unified_score(candidate, nc, platform_profile, audience_context, scoring_mode):
    nc_result = nc_score(candidate, nc, mode="unit")

    intrinsic_score = candidate["metadata"].get("intrinsic_score", 0.0)
    platform_fit_score = candidate["metadata"].get("platform_fit_score", 0.0)
    audience_fit_score = candidate["metadata"].get("audience_fit_score", 0.0)
    risk_penalty = candidate["metadata"].get("risk_penalty", 0.0)

    weights = UNIFIED_SCORE_MODE_WEIGHTS[scoring_mode]

    total_score = max(
        0.0,
        min(
            1.0,
            weights["intrinsic"] * intrinsic_score
            + weights["nc"] * nc_result["total_score"]
            + weights["platform"] * platform_fit_score
            + weights["audience"] * audience_fit_score
            - weights["risk"] * risk_penalty,
        ),
    )

    hard_fail_reasons = list(nc_result["hard_fail_reasons"])
    hard_constraint_pass = len(hard_fail_reasons) == 0

    return UnifiedScoreResult(
        total_score=total_score,
        intrinsic_score=intrinsic_score,
        nc_alignment_score=nc_result["total_score"],
        platform_fit_score=platform_fit_score,
        audience_fit_score=audience_fit_score,
        risk_penalty=risk_penalty,
        hard_constraint_pass=hard_constraint_pass,
        hard_fail_reasons=hard_fail_reasons,
        score_version="unified_score_v1",
        mode=scoring_mode,
    )
```

## III.2 Canonical metrics

### NC_COHERENCE

Per output:

\[
NC\_COHERENCE(u)=NCScore(u,NC)
\]

Definition:

```text
how aligned one output is to the active Natural Center
```

### NC_CONSISTENCY

Per campaign:

\[
NC\_CONSISTENCY(C)=1-\frac{1}{|C|}\sum_{u\in C}D(u,\mu_C)
\]

Definition:

```text
how internally coherent a set of outputs is
```

### NC_DRIFT

Across time:

\[
NC\_DRIFT(NC_t,NC_{t+1})=D(NC_t,NC_{t+1})
\]

Definition:

```text
how much the active Natural Center changed between versions
```

### NC_RECOGNITION

Audience-facing:

\[
NC\_RECOGNITION=f(repeat\_engagement, recall, branded\_search, symbolic\_association)
\]

Definition:

```text
whether the audience is actually recognizing the identity field
```

### NC_SYMBOLIC_DENSITY

Per output:

\[
NC\_SYMBOLIC\_DENSITY(u)=\frac{matched\_markers}{possible\_markers}
\]

### NC_NEGATIVE_SPACE_RISK

Per output:

\[
NC\_NEGATIVE\_SPACE\_RISK(u)=\Omega_{viol}
\]

### NC_TONAL_DRIFT

Per output:

\[
NC\_TONAL\_DRIFT(u)=1-\Tau_{align}
\]

\[

\[


