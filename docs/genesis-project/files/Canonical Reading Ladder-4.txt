# VI. NC ALIGNMENT SURFACE

This is the most important addition.

Natural Center alignment cannot remain a single opaque cosine score. It must be decomposed.

Define:

\[
N(x,NC)=
\lambda_T T_{align}
+\lambda_A A_{align}
+\lambda_{\Tau} \Tau_{align}
+\lambda_N N_{align}
+\lambda_S S_{align}
-\lambda_{\Omega}\Omega_{viol}
\]

where:

```text
T_align      = thematic alignment
A_align      = aesthetic alignment
Tau_align    = tonal alignment
N_align      = narrative alignment
S_align      = symbolic marker density / recurrence
Omega_viol   = negative-space violation score
```

This directly operationalizes the full NC object rather than reducing it to a flat embedding comparison. ŅłĆfileciteŅłéturn4file16Ņłü ŅłĆfileciteŅłéturn4file19Ņłü

### Practical V1 formula

```python
def nc_score(output, nc):
    return clamp(
        0.22 * thematic_alignment(output, nc)
      + 0.18 * aesthetic_alignment(output, nc)
      + 0.18 * tonal_alignment(output, nc)
      + 0.14 * narrative_alignment(output, nc)
      + 0.18 * symbolic_alignment(output, nc)
```

# VII. METRIC SURFACE

The source set explicitly says NC must be independently measurable, not merely used as a modifier. ŅłĆfileciteŅłéturn4file19Ņłü

So define four first-class metrics.

## VII.1 NC Coherence

Per output:

\[
NC\_coherence(u)=N(u,NC)
\]

Meaning: how aligned one output is to the current Natural Center.

## VII.2 NC Consistency

Per campaign:

\[
NC\_consistency(C)=1-\frac{1}{|C|}\sum_{u \in C} D(u,\mu_C)
\]

Meaning: how mutually coherent outputs are with each other under the same center.

## VII.3 NC Drift

Across time:

\[
NC\_drift(t_1,t_2)=D(NC_{t_1},NC_{t_2})
\]

Where \(D\) is the mixed symbolic-vector distance proposed in the proof bridge. ŅłĆfileciteŅłéturn4file24Ņłü

## VII.4 NC Recognition

Audience-facing, later-stage metric:

\[
NC\_recognition = f(recall,\ repeat\_engagement,\ branded\_search,\ symbolic\_association)
\]

This can start as null or experimental until instrumentation matures.

\[
NC_{t+1} = NC_t + \Pi_{\epsilon}\bigl(\Delta_{validated}\bigr)

where \(\Pi_{\epsilon}\) is a projection onto the bounded-drift ball.

Operationally:

```python
def update_natural_center_v2(nc_t, validated_signals, epsilon):
    candidate = apply_validated_updates(nc_t, validated_signals)
    if distance(candidate, nc_t) > epsilon:
        candidate = project_back(candidate, nc_t, epsilon)
    return candidate
```
